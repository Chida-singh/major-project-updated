# Missing / Not-Committed Assets Checklist

This repo intentionally does **not** include large datasets, videos, or private keys. Use this checklist to set up a new machine.

The source-of-truth roadmap is `docs/ROADMAP.md`.

## Secrets

- [ ] `OPENAI_API_KEY` — required for GPT‑4o translation (task `13_set_openai_api_key`).
  - Store in environment variables (do not commit).

## Datasets (large)

- [ ] WLASL repo clone at `WLASL/` (task `02_download_wlasl_repo`)
  - Expected metadata path: `WLASL/start_kit/WLASL_v0.3.json`
  - Expected videos path: `WLASL/start_kit/videos/*.mp4`
  - Not committed.

- [ ] ASL Citizen dataset (task `03_download_asl_citizen_dataset`)
  - Location: choose a local folder (not committed).
  - A conversion script will be needed to unify into the same pose DB format.

- [ ] How2Sign dataset access + downloads (task `01_apply_how2sign_access` + `12_add_how2sign_pkls`)
  - Your current folder: `D:\DSCE\Major Project\how2sign_pkls_default_shape\how2sign_pkls_cropTrue_shapeFalse`
  - Notes:
    - Files are Torch-serialized and may reference `cuda:0`.
    - Requires a PyTorch-compatible Python (typically 3.10–3.12).
