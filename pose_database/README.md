# `pose_database/`

Local pose database generated from datasets like WLASL.

## What’s in here

- `index.json` — list of known glosses / vocabulary manifest.
- `*.npy` — per-gloss pose arrays (large; typically not committed).

## Notes

- The DB is generated by `tools/pose_pipeline.py build-db`.
- `.npy` files are intentionally ignored by git; treat this folder as a build output.
